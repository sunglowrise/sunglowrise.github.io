使用pip安装自己的程序（上传python程序到PyPi）

1. 注册Pypi账号

注册：https://pypi.org/account/register/

2. 安装 setuotools, wheel, twine

setuotools 和 wheel 用来构建项目，一般都会随 Python 安装。

twine 用来上传自己的包到PyPi

3. 构建项目

  • 项目结构

  • setup.py文件

    setup.py 是setuptools的构建脚本。包含了自己的包（例如名称和版本）以及要包含的代码文件。

    重要参数说明

参数 说明
name 包名，发布之前请上 PyPi 搜索一下有没有同名的包，防止冲突。
version 版本号，更新时会寻找比当前版本更高的版本号，不要乱写。
description 短描述
long_description 长描述，详细的介绍。
url 项目的地址，一般会填 github 地址。
packages 包列表，setuptools.find_packages() 可以自动找到目录下所有的包。
install_requires 该包的所需的依赖。
classifiers 分类，根据 PyPi Classifiers 填写，至少要包含所用的 Python 版本。
  • setup.cfg文件

4. LICENSE

上传到 PyPi 的每个包都要包含许可证，选择许可证，打开 LICENSE 并输入许可证文本。
